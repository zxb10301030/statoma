# Asset Credits

This file records the source and license status of Statoma visual assets.

## Site Identity Assets

- `app/icon.tsx`: self-created Statoma monogram generated with Next.js image metadata routes. No external image, stock image, or third-party asset is used.
- `app/apple-icon.tsx`: self-created Statoma monogram generated with Next.js image metadata routes. No external image, stock image, or third-party asset is used.

## Open Graph Images

- `app/opengraph-image.tsx`: self-created Statoma brand Open Graph image generated in code. No external image, stock image, or third-party asset is used.
- `app/calculators/[slug]/opengraph-image.tsx`: self-created calculator Open Graph images generated in code from Statoma calculator metadata. No external image, stock image, or third-party asset is used.

## Notes

Statoma does not currently use raster photography, AI-generated stock images, or third-party icon artwork in public page assets.
